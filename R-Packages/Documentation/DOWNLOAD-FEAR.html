Before using FEAR, one must download and install R. The R installation program and documentation for R can be found at the R website. R is distributed under the GNU General Public License Version 2. After installing R, the FEAR software can be installed into R; see the FEAR Installation Instructions for details.

Please note that the proper citation to acknowledge that you have used FEAR in your research is to cite

    Wilson, Paul W. (2008), "FEAR 1.0: A Software Package for Frontier Efficiency Analysis with R," Socio-Economic Planning Sciences 42, 247--254.


If you are upgrading R or FEAR from a previous version for which you have a license, be sure to save the 'license.dat' file from the older installation. You can find the directory containing this file by starting R and typing 'find.package("FEAR")'.
